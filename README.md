# DayForge

DayForge is an open-source personal operating system built around daily planning, daily review, finance tracking, study tracking, gym tracking, and long-term behavioral analytics.

The goal is simple: reduce the friction of self-tracking by using a Telegram bot instead of spreadsheets.

## Core idea

Plan tomorrow. Review today. Track reality. Learn patterns.

## MVP Modules

- Daily planning
- Daily review
- Finance tracking
- Study tracking
- Gym tracking
- Weekly summaries

## Tech Stack

- Node.js
- TypeScript
- grammY
- PostgreSQL
- Prisma
- Docker
